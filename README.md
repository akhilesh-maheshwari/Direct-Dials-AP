# 📞 Direct Dials Scraper

Get verified direct dial phone numbers for any LinkedIn profile. Simply provide LinkedIn profile URLs as input, and receive accurate phone numbers for your prospects. Designed for sales outreach, cold calling, recruiting, and lead enrichment workflows.

We are the best provider for direct dials in the market — quality is top-notch. Test it out yourself by submitting a batch of 100 LinkedIn profiles.

**No login cookies required.**

---

## 📌 Data Fields Included

| Field | Description |
|---|---|
| linkedInUrl | LinkedIn profile URL submitted as input |
| phoneNumber | Direct dial phone number found for the profile (empty if not found) |
| status | Match status of the lookup — `"found"` or `"not_found"` |
